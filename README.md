# MATCAP1

MATCAP1 (KIAA0895L), or microtubule associated tyrosine carboxypeptidase 1, is a gene that codes the protein by the same
name, which controls microtubule dynamics and their functions.  This Rosetta project studies the feasibility of
utilizing MATCAP1 to suppress the proliferation of pancreatic cancer cells, through studying data and resources including
NCBI, GEO Profiles, Human Protein Atlas and STRING.
